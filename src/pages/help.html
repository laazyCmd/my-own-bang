How to change hostname?

The hostname is basically a URL address.

From your preferred hostname, take the highlighted part of this example URL: https://duckduckgo.com/, or you can include the domain prefix: https://www.google.com/.

This is required, for this is where the extension will read the search parameters, like the https://duckduckgo.com/?q=!piped+example+query.

How to add bangs?

Adding bangs require a bang name and an address for redirection.

1. Click on the "Add" button, and a new, temporary card will be shown to you.

2. After clicking the button, you are required to add a name for the bang.

3.1 Under the name, you are required to add the shortcut address, you can also add a {{s}} to the address if you wish a string be replaced by query parameters. This shortcut will be used to redirect you.

3.2 If you have query parameters included, the extension will replace a string in the shortcut address through {{s}}: If you type !piped example query, the address https://piped.kavin.rocks/results?search_query={{s}} will become https://piped.kavin.rocks/results?search_query=example query instead.

This will be the format of your bang: ![BANG NAME] [QUERY PARAMETERS].
